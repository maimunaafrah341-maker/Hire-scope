# 🔍 HireScope
### Hiring Risk & Blindspot Detector

AI-powered hiring agent that detects risks human recruiters miss.

## How it works
1. Enter a candidate's LinkedIn URL
2. Bright Data scrapes their real profile dynamically
3. Featherless AI analyses in 3 steps — profile summary, 
   risk detection, hire/no-hire decision

## Tech Stack
- Bright Data — live LinkedIn data scraping
- Featherless AI — open-source LLM reasoning
- Streamlit — interactive dashboard
- Python — core logic

## Problem Statement
Hiring Risk & Blindspot Detector — Forge Inspira 2026 Codeathon

## Setup
1. Clone the repo
2. Add your API keys to .env file
3. Run: streamlit run app.py
